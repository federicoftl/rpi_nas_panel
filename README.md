# Raspberry Pi 3B NAS control panel
 A control panel for the Raspberry controlled by a "5 way + 2 buttons Switch" and a 0.91 OLED Display

Menu composition:
 - Mount partitions 	
	 - Submenu with mountable partitions  
		 - Confirmation screen   
 - Connection
	 - TO DO
 - NAS info
 - Battery info
 - Screen off
 - Reboot
	 - Confirmation screen   
 - Shutdown
	 - Confirmation screen
